# 2022_Media_OpenGL
2022年度メディア情報学実験OpenGL課題

コンパイル方法
1. OpenGLとGLUTをインストール
    - Windowsなら：http://freeglut.sourceforge.net/
2. `g++ prog1 myModel.cpp -lglut –lGL -lGLU` でコンパイル
